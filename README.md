# Loja Online Java

Sistema de loja online desenvolvido em Java puro com foco em Programação Orientada a Objetos, regras de negócio e manipulação de dados em memória.

---

## 🚀 Tecnologias utilizadas

- Java
- Collections Framework
- Streams API
- Optional
- Generics
- Programação Funcional

---

## 📚 Conceitos praticados

- Programação Orientada a Objetos (POO)
- Arquitetura em camadas
- Repositório genérico
- Manipulação de listas em memória
- Streams API
- Filtros e ordenações
- Optional
- Regras de negócio
- Encapsulamento
- Predicates
- Separação de responsabilidades

---

## 📁 Estrutura do projeto

```bash
src

│   ├── Produto.java

├── repositorio

---

## ⚙️ Funcionalidades

* ✅ Cadastro de produtos
* ✅ Cadastro de clientes
* ✅ Cadastro de pedidos
* ✅ Busca de clientes por CPF
* ✅ Filtro de produtos por categoria
* ✅ Listagem de produtos ativos
* ✅ Ordenação de produtos por preço
* ✅ Verificação de produtos gratuitos
* ✅ Listagem de pedidos pagos
* ✅ Cálculo de total do pedido
* ✅ Cálculo de faturamento total
* ✅ Relatório de pedidos por cliente

## ▶️ Como executar o projeto

### 1. Clonar o repositório

git clone <github.com/JKSJ15/LojaOnline>
```

### 2. Abrir o projeto na IDE

* IntelliJ IDEA
* Eclipse

### 3. Executar a classe principal

```bash
LojaService.java
```
